Elegant and powerful network monitoring for iOS and macOS.

## Core Features

- **Real-time Monitoring** — Continuous ping monitoring for multiple servers with live latency display
- **Smart Visualization** — Beautiful charts showing latency trends, supporting 10-minute, 1-hour, and 24-hour time scales
- **Traceroute** - Visualize network paths, showing each hop node along with its latency
- **Long-term Monitoring** — Support up to 30 days of continuous monitoring with complete data point recording
- **History Export** — Export as .pgr format (reopenable for chart viewing) or .csv format (for data analysis)
- **Multi-server Management** — Monitor multiple targets simultaneously with custom icons and names
- **Native Dual Platform** — Native iOS and macOS apps with deep platform-specific optimizations

## Use Cases

- **Daily Use** — Check latency from your iPhone to iPad, or test any local network device
- **Home Network** — Monitor routers, smart TVs, NAS, and other home devices
- **Gaming** — Real-time monitoring of game server latency to optimize experience
- **Server Operations** — Monitor server availability and response times
- **Network Debugging** — Diagnose network issues and locate fault nodes
- **Long-term Analysis** — Up to 30 days of continuous monitoring to catch rare issues

## Privacy First

Your data stays on your device, always.

- All data stored only on your device, completely offline
- No user data collected or uploaded to any server
- No iCloud sync to ensure data security and control
- No third-party tracking or advertisements
- Only accesses the target servers you specify during monitoring
