Run Windows 11 natively on your Apple Silicon Mac — no setup wizard, no configuration panels, no subscription.

## How It Works

1. **Download** WinRun from the App Store
2. **Point** it to a Windows 11 ARM ISO from Microsoft
3. **Done** — WinRun installs everything automatically

From then on, open WinRun and you're at the Windows desktop in seconds. Close the window and your session is saved instantly — pick up right where you left off.

## Key Features

- **Zero Configuration** — No virtual machine to create, no settings to tweak. One app, one Windows.
- **Pay As You Go** — Top up only when you need more. No subscriptions, no upfront cost.
- **Instant Resume** — Close the window to save, open to restore. Your Windows session is always right where you left it.
- **Privacy Friendly** — Runs entirely on your Mac. No accounts, no cloud, no data leaves your device.
- **Native Performance** — Built on Apple's Hypervisor framework with dynamic CPU and memory allocation.
- **App Store Sandboxed** — Full macOS sandbox for system-level security.

## Pay As You Go

WinRun charges by usage time, not by month.

- **Top up 180 minutes** anytime via a single In-App Purchase
- Timer only runs while Windows is active — close the lid and it stops
- Balance shown in the title bar at all times

No subscription. No expiry. Use it once a week or every day — you only pay for the time you use.

## System Requirements

- Apple Silicon Mac (M1 or later)
- macOS 26.0 or later
- Windows 11 ARM ISO (download free from Microsoft)
