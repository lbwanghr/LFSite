Run a headless Windows 11 virtual machine on your Apple Silicon Mac — access it anytime via Remote Desktop.

## How It Works

Download a Windows 11 ARM ISO from Microsoft, point SubWin to it, and wait for the automated installation to finish. Once ready, connect to `localhost:3389` with any RDP client — Microsoft's free [Windows App](https://apps.apple.com/app/windows-app/id1295203466) works great.

SubWin lives in your menu bar, out of your way. No windows, no dock icon — just a lean VM that's always there when you need it.

## Key Features

- **Menu Bar App** — Runs quietly in your menu bar. No windows, no dock clutter.
- **Remote Desktop Access** — Connect via RDP at `localhost:3389` using any RDP client.
- **One-Time Purchase** — Pay once, use forever. No subscriptions, no usage limits.
- **Privacy Friendly** — Runs entirely on your Mac. No accounts, no cloud, no data leaves your device.
- **Native Performance** — Built on Apple's Hypervisor framework with sensible CPU and memory allocation tuned to your Mac.
- **App Store Sandboxed** — Full macOS sandbox for system-level security.

## System Requirements

- Apple Silicon Mac (M1 or later)
- macOS 26.0 or later
- Windows 11 ARM ISO (download free from Microsoft)
